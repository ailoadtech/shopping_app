# Shopping List App

A simple Android shopping list application built with Jetpack Compose and Material Design 3.
I needed an offline app reflecting all my supermarkets where i am buying, so i build it.

## Features

- **Add Items**: Enter item names via text input field
- **Swipe to Delete**: Swipe items left or right to remove them
- **Filter by Store**: Filter your shopping list by supermarket (Rewe, Hit, DM, Mueller, Edeka, Kaufland)
- **Persistent Storage**: Items are automatically saved to local file storage
- **No Permissions Required**: Uses app-specific storage, no Android permissions needed
- **Material 3 UI**: Modern Android UI with Material Design 3 components

## Quick Start

### Prerequisites

- Android SDK (API 34 recommended)
- Android Studio (or Gradle installed)
- Java Development Kit (JDK 17+)

### Building the App

```bash
# Build debug APK
./gradlew assembleDebug

# Build release APK
./gradlew assembleRelease

# Run tests
./gradlew test

# Install on connected device
./gradlew installDebug
```

### APK Location

After building, the debug APK can be found at:
```
app/build/outputs/apk/debug/app-debug.apk
```

## File Format

Items are stored in app-specific storage: `Android/data/com.example.shoppinglist/files/Download/shoppinglist.txt`

Format:
```
storeNumber,itemName
```

Examples:
```
1,Milk
2,Bread
,Eggs          # unassigned item (no store)
3,Cheese       # item name without commas
```

Store Numbers:
| Number | Store |
|--------|-------|
| 1 | Rewe |
| 2 | Hit |
| 3 | DM |
| 4 | Mueller |
| 5 | Edeka |
| 6 | Kaufland |

## Usage

1. **Adding Items**: Type an item name in the input field and tap OK
2. **Filtering**: Tap a supermarket logo to show only items from that store, or "All" to show everything
3. **Deleting**: Swipe an item left or right to delete it
4. **Note**: New items are always added as "unassigned" (no store). Use edit to assign a store if needed.

## Troubleshooting

### Build Issues

If Gradle fails to sync:
1. Ensure Android SDK is installed and path is set in `local.properties`
2. Run `./gradlew clean` then try building again
3. In Android Studio: File → Sync Project with Gradle Files
